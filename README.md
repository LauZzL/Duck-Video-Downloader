#### 开发环境

- Node.js
- Vue3
- Vite
- Python 3

下载

```bash
git clone https://github.com/LauZzL/Duck-Video-Downloader/Duck-Video-Downloader.git
```

安装依赖

```bash
# 安装python依赖
pip install -r requirements.txt
# 安装node依赖
cd renderer
yarn install
```

运行

```bash
# 运行ui
cd renderer
yarn dev
# 运行python
cd ..
python app.py
```
